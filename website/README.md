# Website

This directory contains Astro stuff for the main `shimmermathlabs.com` website.

Basic commands:

```bash
npm run ...

  dev
    astro dev
  build
    astro build
  preview
    astro preview
  astro
    astro
  deploy
    rsync --archive dist/ root@shimmermathlabs.com:/var/www/shimmermathlabs/
```
